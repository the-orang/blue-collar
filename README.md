# blue-collar
Rivals of Aether workshop character. Should've started this repo months ago.

# Installing
You can install Blue Collar by checking out his [Steam workshop page](https://steamcommunity.com/sharedfiles/filedetails/?id=3494473915), or if you want to edit your own version you can grab the ```.zip``` from the Releases tab and move ```blue_collar``` into ```%localappdata%\RivalsofAether\workshop\```. From there he should show up in the Workshop menu, without a download icon. You can edit him by following the guide for GMEdit [here at Rivalslib](https://www.rivalslib.com/workshop_guide/quickstart/software_setup.html#gmedit-code-editor).
